# Summarization Example
  
## Example Card
  
- Dataset
  - Name: XSum
  - Language: English
  - Data Size: 0.2M  
  - Link: [[link]](https://huggingface.co/datasets/xsum)
- Model
  - Name: Bart
  - Language: English
  - Link: [[link]](https://huggingface.co/facebook/bart-base)  
- Requirements
  - pytorch
  - huggingface
  - wandb
  - datasets
  - rouge
  
## Get Started
  
You can train with [KoWiki corpus](https://github.com/lovit/kowikitext) directly.
  
- Train

```
$ python3 train.py
```
  
- Play with model
  
```
$ python3 play.py
```
